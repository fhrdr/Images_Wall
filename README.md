# Images_Wall - 图片展示小工具

![License](https://img.shields.io/badge/license-MIT-blue.svg)

一个基于 Node.js 的本地图片展示工具，可以浏览文件夹中的所有图片并以网格形式展示。

## 功能特点

- 📁 **层级文件夹浏览**：支持多层嵌套文件夹结构，逐级浏览
- 🖼️ **图片展示**：自动识别并展示多种格式的图片文件（jpg, jpeg, png, gif, bmp, webp）
- 🌐 **Web界面**：通过浏览器访问，友好的用户界面
- 📱 **响应式设计**：适配不同屏幕尺寸
- 🚀 **简单易用**：一键启动，无需复杂配置

## 快速开始

### 启动方式

1. **使用命令行启动**：
   ```bash
   node server.js
   ```

2. **使用启动脚本**（Windows）：
   ```bash
   start.bat
   ```

### 访问应用

启动服务器后，在浏览器中访问：
```
http://localhost:3000
```

## 使用说明

1. **首页浏览**：
   - 进入首页后，会显示根目录下的所有文件夹
   - 点击文件夹图标进入该文件夹

2. **层级浏览**：
   - 进入文件夹后，会显示该文件夹下的子文件夹和图片
   - 点击子文件夹可继续深入浏览
   - 使用面包屑导航可以查看当前路径
   - 点击".."按钮或使用面包屑导航可返回上级目录

3. **图片查看**：
   - 图片以网格形式展示
   - 自动适配图片大小
   - 支持中文文件名和文件夹名

## API 接口

| 接口地址 | 功能说明 |
|---------|---------|
| `/api/directories` | 获取根目录下的文件夹列表 |
| `/api/subdirectories/{folder}` | 获取指定文件夹下的子目录列表 |
| `/api/images` | 获取根目录下的图片列表 |
| `/api/images/{folder}` | 获取指定文件夹下的图片列表 |
